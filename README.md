## Java Developer Study Guide

Lab exercises for fundamental concepts in Java SE 21 in preparation for developer certification.

## Labs

- `lab 01`: object-oriented concepts
- `lab 02`: handle text and numeric variables
- `lab 03`: array sort and comparable interface
- `lab 04`: handle errors and exceptional conditions
- `lab 05`: inheritance, subtyping between subclass and superclass
- `lab 06`: polymorphism and interfaces
